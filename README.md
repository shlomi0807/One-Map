# Interactive One Piece World Map 🏴‍☠️🗺️

## About The Project
An interactive, web-based map of the One Piece world. This project allows users to explore the Grand Line and beyond, find their favorite characters, and view real-time status updates and bounties. It features smooth panning, zooming, and a dynamic search engine that navigates directly to specific characters on the map.

This project was developed as part of a third-year academic course, showcasing frontend development skills including DOM manipulation, event handling, data structure management, and modern CSS layout techniques.

## Key Features
* **Interactive Exploration:** Smooth dragging, panning, and zooming across a massive high-resolution map (powered by Panzoom).
* **Smart Search:** A dynamic sidebar search that instantly locates characters and automatically flies the camera to their exact coordinates.
* **Dynamic Markers & Popups:** Clickable character markers that reveal custom information windows (status, bounty, current location).
* **Custom UI:** A sleek, collapsible sidebar navigation menu with custom icons.
* **Data Separation:** Character data is managed independently via a JavaScript data structure for easy maintenance and scalability.

## Built With
